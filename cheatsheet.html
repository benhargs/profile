HTML



Hyper Text Mark-Up Language is the building blocks for webpage design.

In order to use the tags below make sure to put <> aroud the desired tag.

Tags
Tag What it does
html The HTML tag is the container for all other HTML elements
Head The head tag contains the title of the document, styles, scripts and other HTML elements. The data's data
Body This defines the document's body.
h1-h6 These establish text getting bigger, h1 being the largest, to h6 as the smallest (being bigger then untagged text).
br Creates a line break.
a The anchor element creates a link. With href attribute it creates a hyperlink to web pages.
div The div tag defines a division on section in an HTML document. It's used as a container for HTML elements.
Meta The meta tag defines metadata about an html document. metadata is infromation about data.

CSS

Cascading Style Sheets is the language to design and make webpages presentable.

Seletor Name What does it select
ID selector id= The element on the page with the specified ID. On a given HTML page, each id value should be unique.
Class selector . The elements on the page with the specified class.
Attribute selector img[src] The element(s) on the page with the specified attribute.
Pseudo-class selector The specified element(s), but only when in the specified state. (e.g. when a cursor hovers over a link.)
